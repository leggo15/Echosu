About echosu

What is echosu

echosu is a tool for people to find specific types of maps, whether it is a large number of a general mapping type or extremely specific maps.

How does it work?

There are two main features to understand: the search function and the tagging system.

Search

The search feature allows you to find maps using any tag or map attribute such as approach rate, circle size, or title. The most powerful aspect of the search engine is the tag operators:

  • Minus symbol (-): Excludes maps with the specified tag.
  • Dot symbol (.): Includes only maps with the specified tag.
  • Quotes (" "): Searches for an exact phrase or a tag consisting of multiple words.

The search results are based on how many of the specific tags you search for are applied to the map. For example:

  • Map A: aim(7), farm(6), "high spacing"(2)
  • Map B: aim(3), farm(6), "bursts"(6)
  • Map C: aim(11), farm(1), "bursts"(4)
If you search for [aim farm "high spacing"], the results would be:
  • Map A (15)
  • Map C (12)
  • Map B (9)
These weights determine the ranking based on tag relevance.

Tags

Maps are labeled by users through the tagging system. These tags represent mapping genres such as aim, speed, streams, tech, stamina, etc. Users can also create new tags to accommodate emerging terms.

There are two methods to apply tags to maps:

  • Clicking on tags already applied to a map.
  • Using the search form found on the home page and the map detail page to search for and apply new tags.
This allows for the addition of both existing and completely new tags.

Tag Descriptions

Tags have descriptions that appear when hovering over them. These descriptions are written by users who have been active enough to access the "Edit Tags" page, available in the dropdown menu alongside settings and the logout button. Access is unlocked after tagging a certain number of maps.

For Developers

echosu generates labeling data for osu maps, which is valuable for analytics and model training. An API has been set up to fetch this data. Documentation is available here. The API is currently not open source due to the need for refactoring. However, requests for repository access can be sent to "The Emperor" on osu or "Leggo15" on Discord.
