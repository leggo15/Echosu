How to use
You can search for specific genres like Aim, Tech and such by typing them in. If you're looking for a genre with spaces in the name you need to encapsulate it using quotes like so; "Spaced Streams". You can also search for specific map characteristics such as AR=8 or an artist- or mapper's name. These rules can be combined into a very specific search. An example could look like: "Aim awkward "No Bursts" AR=9 Od>8 cs<6 Monstrata".
